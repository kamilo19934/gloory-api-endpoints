PUT
/citas/


Este endpoint permite cambiar el estado de una cita.

Condiciones

Citas en estado eliminado no se pueden modificar.
Citas en estado suspendido no se pueden confirmar.
Citas en estado lista de espera no se pueden confirmar o suspender.
Los estados por medio de la API se pueden modificar a confirmar (C) o suspendido (S).
Parameters
Try it out
No parameters

Request body

application/json
Examples: 
Ejemplo de body
Example Value
Schema
{
  "uuid": "876876dsad868868d67a8f86f8ao",
  "estado_codigo": "C"
}
Example Description
Ejemplo de body correcto, para el caso de estado_codigo, este puede tener el valos de C(confirmado) o S(suspendido), otros valores llevaran al error.
